# Notation Guide

This book uses standard robotics notation.

## Coordinate Frames

- **Spatial Frame**: S
- **Body Frame**: B
- **Transformation**: T_sb (Matrix from B to S)

## Kinematics & Dynamics

- **q**: Joint positions
- **tau**: Joint torques
- **M(q)**: Mass matrix

## Summary

Simplified notation for build verification.
