# Implementation Plan: Book Redesign & UX Overhaul

**Branch**: `005-book-redesign` | **Date**: 2025-12-19 | **Spec**: [specs/005-book-redesign/spec.md](spec.md)

## Summary

This plan outlines the redesign of the AI-Native Textbook to improve navigation, standardize chapter structure, and add auxiliary educational resources. The primary technical changes involve configuring Docusaurus for autogenerated sidebars, setting the Intro module as the site root, and implementing standalone Glossary and Notation pages using MDX.

## Technical Context

**Language/Version**: Node.js >= 20.0, TypeScript 5.x
**Primary Dependencies**: Docusaurus 3.9.2, React 19, remark-math, rehype-katex
**Storage**: Git-based MDX files
**Testing**: Manual validation, `tsc`, `validate-content.js`
**Target Platform**: GitHub Pages
**Project Type**: Docusaurus Static Site (Web)
**Performance Goals**: Standard Static Site performance (Lighthouse > 90)
**Constraints**: Must maintain existing content structure; deployable to GitHub Pages.
**Scale/Scope**: ~8 Modules, <100 Chapters.
