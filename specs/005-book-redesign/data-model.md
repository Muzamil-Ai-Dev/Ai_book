# Data Model: Book Redesign

## File System Structure

```text
apps/web/
├── docusaurus.config.ts    # Main config (routeBasePath: '/')
├── sidebars.ts             # Sidebar config (autogenerated)
├── src/
│   ├── css/
│   │   └── custom.css      # Global styles
│   └── pages/
│       ├── glossary.md     # Standalone Glossary Page
│       └── notation.md     # Standalone Notation Guide
└── modules/                # Content Source (was 'docs')
    ├── 01-intro/           # Module 01
    │   ├── index.md        # Intro Page (slug: /)
    │   └── ...
    ├── 02-ros2/            # Module 02
    │   ├── index.md        # Module Overview
    │   ├── 01-nodes.md     # Chapter
    │   └── ...
    └── ...
```

## Content Schema (MDX)

### Chapter (`modules/**/*.md`)

**Frontmatter:**
```yaml
title: [String, Required] - Chapter Title
description: [String, Required] - SEO description
sidebar_label: [String, Optional] - Short name for sidebar
sidebar_position: [Integer, Optional] - Ordering
```

**Structure:**
1.  Heading 1 (`# Title`)
2.  Heading 2 (`## Learning Objectives`)
3.  Heading 2 (`## Prerequisites`)
4.  Heading 2 (`## Core Concepts`)
5.  Heading 2 (`## Examples / Exercises`) OR (`## Examples`) OR (`## Exercises`)
6.  Heading 2 (`## Summary`)
7.  Heading 2 (`## References`)

### Glossary (`src/pages/glossary.md`)

**Frontmatter:**
```yaml
title: Glossary
description: Physical AI & Robotics Terminology
hide_table_of_contents: false
```

**Structure:**
*   Heading 1 (`# Glossary`)
*   Alphabetical sections (`## A`, `## B`, ...)
*   Terms as bold items (`**Term**: Definition`)

### Notation (`src/pages/notation.md`)

**Frontmatter:**
```yaml
title: Notation Guide
description: Mathematical & Robotics Notation
```

**Structure:**
*   Heading 1 (`# Notation Guide`)
*   Tables defining symbols using KaTeX (e.g., `$T_{eb}$`)
